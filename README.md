### Iniciar projeto:

Web:
```
npm run dev
```

Server:
```
npm run dev
```

### Tecnologias

- React
- Node
- Typescript
- Prisma
- Express
- CORS
- Vite
- Tailwind

### API:

- Listagem de Games (GET)
- Listagems de Anúncios por Game (GET)
- Buscar código Discord por Anúncio (GET)
- Criar Anúncios (POST)
